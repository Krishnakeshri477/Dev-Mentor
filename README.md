# DevMentor AI 🚀

**DevMentor AI** is an AI-powered career platform providing personalized mentorship, resume analysis, and job matching for developers.

---

## ✨ Features
- **🧠 AI Resume Analysis**: Get ATS scores.
- **🎯 AI Talent Matcher**: Match your skills to real-time job listings.
- **💬 AI Career Coach**: Context-aware AI for technical and career advice.
- **📊 Intelligence Dashboard**: Visualize your progress and coaching gaps.
- **🌗 Dark Mode**: Premium, glassmorphic UI with full theme support.

---

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS, Recharts, Axios.
- **Backend**: Node.js, Express, MongoDB, Groq AI.
- **Services**: Google OAuth, ImageKit, Hindsight Memory.

---

## 🚀 Quick Start

### 1. Backend
```bash
cd backend && npm install
# Setup .env with: PORT, MONGO_URI, GROQ_API_KEY, JWT_SECRET
npm start
```

### 2. Frontend
```bash
cd frontend && npm install
# Setup .env with: VITE_API_URL
npm run dev
